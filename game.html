Les règles du Shifumi:

la sorcière empoisonne le loup-garou

la sorcière blasphème l’ange

le loup garou mange les villageois

le loup-garou blasphème l’ange

le démon corrompt la sorcière

le démon asservi le loup-garou

l'ange renvoi en enfer le démon

l’ange anathématise les villageois

les villageois conjurent le démon

les villageois brulent la sorcière
